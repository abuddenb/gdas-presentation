What is GDAS?

A web application to facilitate the rapid entry and processing of traceable, sound global change data, information, and products.

What is GDAS?

A place to collaborate on metadata in real time.

What is GDAS?

The answer to a question:

What if we made metadata fast and easy?

Why did we make GDAS?

The Assessment has a lot of moving parts.

Why did we make GDAS?

A single place to coordinate our activities that's always up-to-date.

vs.

  • Emailing works-in-progress among geographically disparate groups. ("Wait, is this the latest version!?")
  • A constellation of Drupal forms, ad-hoc spreadsheets, a Google Slides presentation, and the honor system to manage metadata. ("Is that on the resources site? Check the status in the spreadsheet. No, there was a problem :-/ ")
  • Sending out an Excel sheet as a metadata survey. (This didn't last long; we got exactly one response.)

Why did we make GDAS?

Metadata shouldn't be this hard!

Many moving parts create friction.

Users shouldn't have to worry about breaking things.

It should be easy to add and change metadata - "Just type."
